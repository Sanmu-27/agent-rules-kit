# Refactor Prompt

```text
Refactor this code without changing behavior.

Before editing:
- identify the current behavior
- inspect call sites
- find existing tests or add characterization tests if risk is high

When editing:
- keep the refactor small and reviewable
- avoid mixing feature changes with cleanup
- preserve public APIs unless explicitly asked otherwise

Before finishing:
- run focused verification
- summarize what changed structurally and confirm behavior stayed the same
```
