# Pull Request Review Prompt

```text
Review this pull request like a senior engineer.

Focus on:
- correctness
- regressions
- security
- data loss
- performance
- missing tests

Return findings first, ordered by severity.
For each finding, include:
- file and line
- impact
- why it matters
- suggested fix

If there are no findings, say so clearly and mention any residual risk.
```
