# Bug Fix Prompt

```text
Investigate this bug in the existing codebase.

Before editing:
- inspect the relevant code paths
- identify likely root cause
- check for existing tests around this behavior

When editing:
- make the smallest change that fixes the root cause
- avoid unrelated refactors
- add or update a regression test if practical

Before finishing:
- run the most relevant verification command
- summarize the root cause, fix, and test result
```
