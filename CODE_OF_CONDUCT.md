# Code Of Conduct

## Our Standard

This project welcomes practical, respectful collaboration.

Good contributions focus on:

- real AI coding agent failure modes
- concise and actionable rules
- useful examples
- respectful disagreement

Unacceptable behavior includes harassment, personal attacks, spam, and contributions that intentionally mislead users.

## Enforcement

Maintainers may edit, close, or remove issues and pull requests that do not follow this standard.

If a discussion gets heated, assume good intent first, then return to concrete examples and reproducible behavior.
